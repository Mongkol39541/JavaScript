National Software Contest (NSC)

ได้เข้าร่วมการแข่งขัน NSC การแข่งขันพัฒนาซอฟต์แวร์ ซึ่งในหัวข้อที่ผมทำเกี่ยวกับการแพทย์และผู้พิการ มีหัวข้อชื่อว่า Sign Language ผ่านเข้ารอบนำเสนอผลงาน และได้รับเงินรางวัล 10,000 บาท เพื่อใช้ในการพัฒนาโปรแกรม โดยภายในกิจกรรมผมทำหน้าที่เป็นคนพัฒนาฝั่ง AI และ Web เป็นหลัก
